# 🔥 ali-forged

> A fine-tuned large language model trained to think, talk, and respond like me.

![Status](https://img.shields.io/badge/status-in%20progress-orange?style=flat-square)

---

## What is this?

Most people use ChatGPT with a system prompt and call it a "personal AI." I didn't want that.
ali-forged is a proper fine-tuned LLM — trained on real data that reflects how I think, what I've built, and how I communicate. The goal is a chatbot on my personal website that genuinely represents me, not a generic AI with my name slapped on it.
This repo documents the full journey: data collection → formatting → fine-tuning → deployment.
